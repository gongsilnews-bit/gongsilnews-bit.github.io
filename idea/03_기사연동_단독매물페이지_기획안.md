# 🚀 향후 개발 기획: 뉴스 기사 연동 '폐쇄형 단독 매물 페이지' 

## 기획 배경
- **상황:** 유료 부동산 회원이 '우리동네뉴스' 기사를 작성하고, 해당 기사 하단(또는 우측)에 본인이 올린 '일반 공개용 매물' 리스트를 노출함.
- **문제점:** 독자가 해당 매물을 클릭했을 때 일반적인 '공실열람(전체 매물 포털)' 페이지로 넘어가게 되면, 독자가 검색을 통해 **타 부동산의 매물을 보고 이탈(손님 뺏김)**할 우려가 매우 큼. 유료 회원(작성자)은 이를 매우 꺼려할 것임.
- **해결책:** 독자를 포털 내에 풀어두지 않고, 오직 기사 작성 중개사의 매물만 볼 수 있도록 '가두리(Walled Garden)' 형태의 단독 페이지로 유도.

## 구현 아이디어

### 1. 팝업창 (또는 화면 덮기 모달) 방식
- 기사 하단의 매물을 클릭하면 페이지가 아예 넘어가지 않고, 현재 화면 위에 큰 **팝업(모달) 창**이 뜸.
- 이 창 안에서 매물 사진, 가격 상세 정보를 확인하고 작성자에게 바로 전화를 걸 수 있음.
- 창을 닫으면 다시 중개사가 작성한 뉴스 기사로 돌아옴. 완전히 고객의 동선을 통제할 수 있는 가장 확실한 방법.

### 2. 폐쇄형 단독 랜딩 페이지 방식 (앞선 아이디어의 연장선)
- 매물 클릭 시 새 창이 열리지만, 일반 공실열람 페이지가 아님.
- 주소창의 파라미터가 `?mode=standalone&agent=xxx` 형태로 됨.
- **특징:**
  - 홈페이지 전체 메뉴(GNB) 없음.
  - 검색창 없음.
  - '다른 부동산 매물 보기' 등의 관련 링크 추천 없음.
  - 오직 '해당 매물 상세 정보' + '이 부동산의 다른 매물 더 보기' + '전화걸기' 버튼만 큼지막하게 존재함.

## 기대 효과
- 유료 회원이 기사를 작성할 **동기부여(마케팅 효과)를 극대화**시킴. (내 기사를 읽은 손님은 무조건 내 손님이 됨)
- 플랫폼에 광고비를 지불하는 중개사에게 확실한 리드(Lead, 잠재고객)를 보장하여 유료 결제의 가치를 높임.
