# 🚀 향후 개발 기획: 공공데이터 API 연동 기반 '아파트 단지 정보 실시간 자동화' 시스템

## 기획 배경
- **현행 문제점:** 부동산 중개사가 매물을 올리거나 고객에게 설명할 때, 해당 아파트 단지의 준공연도, 총 세대수, 주차 대수, 난방 방식, 최근 실거래가 등을 일일이 네이버나 다른 포털에서 찾아보고 수기로 입력해야 하는 번거로움과 오타율 발생.
- **해결책:** 대한민국 정부(국토교통부)에서 무료로 제공하는 **'공공데이터포털(data.go.kr) 오픈 API'** 서버와 공실뉴스 서버를 직접 파이프라인으로 연결하여, 단지 정보와 실거래가를 0.1초 만에 자동으로 끌고 오는 기능 구축.

## 구체적 기술 적용 부분 (사용 API)

### 1. 국토교통부 _ 공동주택 기본 정보 API
- **동작 방식:** 관리자 페이지에서 부동산 소장님이 매물의 아파트명(단지명)을 검색하거나 입력함.
- **자동 호출 데이터:** 
  - 총 세대수 / 동 수
  - 사용승인일(준공연도)
  - 총 주차 대수 (세대당 주차 대수 계산 자동화)
  - 난방 방식 (개별/지역 등)
  - 시공사 및 관리사무소 연락처
- **적용:** 이를 통해 중개사는 단지 정보를 단 1글자도 타이핑할 필요 없이, 가장 최신의 '국가공인' 관공서 데이터를 플랫폼에 박아 넣을 수 있음.

### 2. 국토교통부 _ 아파트매매 실거래자료 API
- **동작 방식:** 등록된 매물의 면적과 매칭하여 국토부에 신고된 과거 거래 내역을 호출.
- **자동 호출 데이터:**
  - 동일 면적의 최근 매매가 / 전월세가 추이
  - 거래 발생 층수 및 거래 일자
- **적용:** 이를 바탕으로 앞서 기획한 **'프리미엄 매매 보고서'**의 핵심 그래프와 시각 자료를 그리는 기초 데이터로 활용.

## 플랫폼의 기대 효과
1. **극강의 편의성 제공:** 부동산 소장님들의 매물 등록 시간이 10분에서 1분으로 단축됨. (플랫폼 충성도 급상승)
2. **법적, 도의적 신뢰성:** "공실뉴스의 데이터 구성은 국가 관공서 데이터와 실시간 연동되어 100% 정확하다"는 타이틀을 획득함.
3. **무비용(Zero Cost):** 이 모든 데이터 연동은 국가에서 민간 프롭테크 기업을 돕기 위해 만든 것이라 **통신 비용이 평생 무료**임. (서버 유지비 절감)
