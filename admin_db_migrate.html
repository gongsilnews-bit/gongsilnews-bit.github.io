🛠️ DB 마이그레이션 - member_no 컬럼 추가

members 테이블에 자동 증가 회원번호 (member_no) 컬럼을 추가합니다.
아래 버튼을 클릭하면 Supabase DB에 직접 적용됩니다.

-- 1. 회원번호(member_no) 추가 ALTER TABLE public.members ADD COLUMN IF NOT EXISTS member_no BIGINT GENERATED ALWAYS AS IDENTITY; CREATE UNIQUE INDEX IF NOT EXISTS members_member_no_idx ON public.members (member_no); -- 2. 유료 멤버십 시작일(membership_start_at) 추가 ALTER TABLE public.members ADD COLUMN IF NOT EXISTS membership_start_at timestamptz; -- 3. 사전 등록 테이블 시작일(start_date) 추가 ALTER TABLE public.pre_registered_realtors ADD COLUMN IF NOT EXISTS start_date date; -- 4. 매물 테이블 중개보수(brokerage_fee) 추가 ALTER TABLE public.properties ADD COLUMN IF NOT EXISTS brokerage_fee text;
✅ 이미 member_no 컬럼이 있으면 무시됩니다 (IF NOT EXISTS)
✅ 기존 회원에게도 자동으로 번호가 부여됩니다
